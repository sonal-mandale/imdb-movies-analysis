# 🎬 IMDB Movies Data Analysis Dashboard

### 🛠️ Tools Used

* SQL (MySQL)
* Power BI

## Project Overview

This project explores IMDB movie data to understand how factors such as popularity, genre, ratings, and directors influence box-office revenue. The interactive Power BI dashboard helps uncover trends and presents insights through clear, business-focused visualizations.

## Key Insights

* Identified the highest-grossing movies and top-performing directors.
* Compared movie popularity with box-office revenue.
* Analyzed genre-wise revenue and rating performance.
* Explored movie release trends across different years.
* Built interactive visuals for filtering by genre and release year.
